
## 工程模块
- spring-ai-chat 聊天模块
- spring-ai-mcp MCP模块
- spring-ai-rag 检索增强(retrieval augmented generation)模块
- spring-ai-toolcalling 工具调用模块

## 工程环境
- spring ai 版本 1.1.8
- Java 版本 17
- Spring Boot 版本 3.5.13

## 编码规范
- 生成的代码（如类、成员变量、关键业务逻辑点）需要加中文注释
- 如果生成的代码是http接口，需要在对应的module下生成响应的测试接口（以".http"为后缀的文件进行承载）
